# Backend Readme

This is the backend for the Yahoo Fantasy Predictions project. It is a Django app that provides a REST API for the frontend to consume.

## Migrations
```
poetry run ./manage.py migrate
```

## Create Superuser
```
poetry run ./manage.py createsuperuser
```
